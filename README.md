# 🩺 Cardiovascular Risk Prediction (ANN)

This project was developed as a clinical decision support tool for cardiovascular risk assessment.

## 🚀 Live Demo
(https://heart-disease-predictor-dim-petr82.streamlit.app/)

## 🛠️ Features
- **Deep Learning Model**: Artificial Neural Network (ANN) trained on the Framingham dataset.
- **Data Engineering**: Outlier handling (Winsorization) and SMOTE for class balancing.
- **Explainability**: SHAP analysis used during development to identify key risk factors.

## 🧪 Tech Stack
- Python (Pandas, Scikit-learn)
- Deep Learning (TensorFlow/Keras)
- Visualization: Plotly (Gauge Charts), Matplotlib, Seaborn
- Deployment (Streamlit Cloud)
